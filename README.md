# EEC-IMX-Generator

`EPlan E104 Generator.xlsm` is an Excel macro workbook used to prepare EPLAN E104 drawing data from InterTrak-style site inputs.

It lets you:
- Load/select a site, rack layout, and I/O points.
- Apply typical card wiring/terminal lookup tables.
- Build card and instrument tag data (including addresses, terminals, and revisions) for EPLAN drawing generation.
